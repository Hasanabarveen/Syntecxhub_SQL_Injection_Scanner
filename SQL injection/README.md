

* SQL Injection Scanner *

1. Description
This project is a Python-based SQL Injection Scanner that detects possible SQL injection vulnerabilities in web applications by testing common SQL payloads.

2. Features
- Tests target URLs for SQL Injection
- Uses predefined payloads
- Generates a scan report

3. Requirements
- Python 3.x
- requests library

4.  Installation
pip install -r Requirements.txt

5. Usage
python scanner.py

6. Output
Results are saved in report.txt

7. Disclaimer
This tool is for educational purposes only. Do not use on unauthorized websites.
